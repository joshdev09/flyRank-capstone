## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/joshdev09/flyRank-capstone.git
   cd flyRank-capstone
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```
